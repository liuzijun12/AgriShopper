# 微信登录功能说明

## 功能概述

本项目实现了完整的微信小程序用户登录功能，用户进入应用后会自动跳转到登录页面，完成登录后可以正常使用应用功能。

## 功能特点

### 1. 自动登录检查
- 应用启动时自动检查用户登录状态
- 已登录用户直接跳转到首页
- 未登录用户跳转到登录页面

### 2. 美观的登录界面
- 渐变背景设计
- 动画效果
- 友好的用户提示
- 隐私政策说明

### 3. 完整的登录流程
- 微信登录凭证获取
- 用户信息获取
- 本地存储管理
- 后端数据同步

### 4. 错误处理
- 网络错误处理
- 用户取消登录处理
- 重试机制
- 友好的错误提示

## 文件结构

```
frontend/
├── pages/

├── utils/
│   └── wxLogin.js          # 登录工具类
├── store.js                 # 状态管理
├── App.vue                  # 应用入口
└── pages.json              # 页面配置
```

## 核心组件

### 1. 登录弹窗组件 (components/WxLoginModal.vue)
- 美观的弹窗UI界面
- 完整的登录逻辑
- 错误处理和重试机制

### 2. 登录工具类 (utils/wxLogin.js)
- `WxLogin`: 微信登录相关方法
- `AuthAPI`: 后端API调用
- `WxLoginManager`: 登录状态管理

### 3. 状态管理 (store.js)
- 用户信息持久化存储
- 全局状态管理
- 数据同步

## 使用流程

### 1. 用户进入应用
```
应用启动 → 检查登录状态 → 已登录？ → 跳转首页
                                    ↓ 否
                                  显示登录页面
```

### 2. 登录流程
```
点击登录按钮 → 获取微信登录凭证 → 获取用户信息 → 保存到本地 → 跳转首页
```

### 3. 错误处理
```
登录失败 → 显示错误提示 → 重试机制 → 达到最大重试次数 → 显示详细错误信息
```

```json
{

  "style": {
    "navigationBarTitleText": "首页"
  }
}
```

### 2. 后端API配置 (utils/auth.js)
```javascript
static baseURL = 'http://localhost:8080/api'
```

### 3. 微信小程序配置
需要在微信小程序后台配置：
- 服务器域名
- 业务域名
- 用户信息获取权限

## 开发注意事项

### 1. 微信环境检查
- 确保在微信小程序环境中运行
- 测试时需要在微信开发者工具中预览

### 2. 网络请求
- 确保后端服务正常运行
- 检查网络连接和域名配置

### 3. 权限配置
- 确保小程序有获取用户信息的权限
- 检查微信开放平台配置

### 4. 测试建议
- 在真机上测试登录流程

## 常见问题

### 1. 登录失败
- 检查微信小程序配置

- 检查后端服务状态

### 2. 页面跳转问题
- 检查pages.json配置
- 确认页面路径正确
- 检查tabBar配置

### 3. 数据存储问题
- 检查本地存储权限
- 确认数据格式正确
- 检查存储空间
